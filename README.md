# Banking AI Agent (packaged)

This folder contains a packaged layout of the project suitable for submission or pushing to a separate repository.

Contents:
- `backend/` — FastAPI backend and its Dockerfile + requirements.
- `intent_service/` — gRPC intent service, proto and generated stubs, server, Dockerfile.
- `frontend/` — Streamlit frontend, Dockerfile + requirements.
- `docker-compose.yml` — compose file to run all services together.

See the top-level README.md for full usage instructions.
